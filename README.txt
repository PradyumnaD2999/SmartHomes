Steps to run the Smart Homes React Application:
1. After downloading the zip file and extracting it, navigate to the directory:
	SmartHomes -> SmartHomes -> smart-homes-backend
2. Open cmd in this "smart-homes-backend" directory.
3. Run the command:
	npm start
4. Navigate to the directory:
	SmartHomes -> SmartHomes -> smart-homes
5. Open cmd in this "smart-homes" directory.
6. Run the command:
	npm start
